👗 Lumière Kebaya
"Elegance in Every Stitch"

Lumière Kebaya adalah website e-commerce yang menyediakan berbagai pilihan kebaya modern, elegan, dan berkualitas premium. Website ini dibuat untuk memenuhi Ujian Akhir Semester Genap mata kuliah Komputer Aplikasi IT-II (E-Commerce/HTML).

🔗 Demo Website: https://atyanabilaa-prog.github.io/Lumi-re-Kebaya/
📂 Repository: https://github.com/atyanabilaa-prog/Lumi-re-Kebaya


📖 Tentang Proyek
Lumière Kebaya menghadirkan koleksi kebaya modern dengan desain elegan, bahan premium, dan pilihan warna pastel yang lembut. Website ini dirancang untuk memudahkan pelanggan menemukan kebaya impian mereka untuk berbagai acara seperti wisuda, lamaran, pesta, dan pernikahan.

Nama "Lumière" berarti cahaya, melambangkan harapan agar setiap pelanggan tampil lebih percaya diri, anggun, dan bersinar saat mengenakan produk dari Lumière Kebaya.

Model Bisnis
Website ini menerapkan model bisnis Business to Consumer (B2C), yaitu menjual produk secara langsung kepada konsumen melalui platform e-commerce tanpa perantara.


Fitur Utama

🏠 Halaman Hero — sambutan awal dengan tombol "Belanja Sekarang"
🛍️ Katalog Produk — menampilkan berbagai model kebaya beserta harga
🔍 Pencarian Produk — mencari produk berdasarkan nama
📋 Detail Produk (Lihat Deskripsi) — pop-up berisi nama, harga, dan deskripsi bahan
🎨 Pilihan Warna — variasi warna untuk setiap produk
🛒 Keranjang Belanja — tambah, ubah jumlah, hapus produk, dan reset keranjang
💳 Checkout & Metode Pengiriman — JNE, J&T Express, SiCepat, AnterAja
💰 Metode Pembayaran — Tunai/COD, QRIS, DANA, ShopeePay, Transfer BNI, SeaBank
🧾 Invoice Pesanan — cetak invoice otomatis setelah checkout
🔐 Login Admin (Dashboard) — kelola stok dan harga produk
